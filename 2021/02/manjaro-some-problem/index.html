Manjaro-DNS等问题.md

  2021-02-24   Linux 0  评论

Manjaro怎么更新DNS呢?默认终端怎么改?怎么在终端下播放音乐?等等一系列问题

更新DNS缓存

查看DNS网址,网址

修改Hosts文件: vim /etc/hosts

刷新dns: sudo systemctl restart NetworkManager.service

各种系统更新DNS缓存的方法->点我出发

Arch有网络,浏览器无法访问

重启网络: systemctl restart systemctl-networkd

Manjaro-gnome设置默认终端

将alaritty设置为我的默认终端,命令: gsettings set org.gnome.desktop.default-applications.terminal exec alaritty

终端下播放音乐

介绍:SOX软件,支持很多格式的音频文件,如 WAV,MP3,MPG,OGG,FLAC 等等.
Arch安装: sudo pacman -S sox

使用方法: play xxx.mp3

Linux符号连接的层数过多

ln软链接用法: ln -s 源文件 目标文件

但是有时候我们软链接文件夹，会提示我们Linux符号连接的层数过多
解决办法:将相对路径替换成绝对路径
