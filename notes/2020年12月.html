Trash命令

日常使用的rm命令没有恢复机制，删除了就找不到了，但是trash这个命令可以让我们将文件放入回收站

  1. 安装 arch下的安装shell sudo pacman -S trash-cli
  2. 使用： Trash-Cli 提供了下面这些命令：
    trash-put==trash： 删除文件和目录（仅放入回收站中）
trash-list ：列出被删除了的文件和目录
trash-restore：从回收站中恢复文件或目录 trash.
trash-rm：删除回收站中的文件
trash-empty：清空回收站
  3. 替代rm 我们可以在环境变量中写入alias rm="trash"来将rm命令更改为trash命令，最后不要忘记刷新全局变量哦！

Git多仓库推送

同时推送到gitee和github

例如
gitee项目地址是:https://gitee.com/liu__yang/clustering-system
github项目地址是https://github.com/Dreamer-liuyang/simple-clustering-system
使用下面的命令关联两个数据库 

git remote add origin https://gitee.com/xueleilei/nuxt-ssr-template.git
git remote set-url --add origin https://github.com/xueleilei/nuxt-ssr-template.git

最后push时会同时push到两个仓库中

不同时推送到gitee和github

git remote add gitee https://gitee.com/monksoul/Fur.git
git remote add github https://github.com/MonkSoul/Fur.git
git push gitee
git push github

Github提交PR

  1. fork到自己仓库
  2. git clone 到本地
  3. 上游建立连接 git remote add upstream 开源项目地址
  4. 创建并切换分支 git checkout -b new-xiaoming
  5. 提交代码
  6. push到自己的仓库中去
  7. 去自己仓库的主页发现多了一个PR的按钮

B站快捷键

  1. ESC/F 这两个都可以退出全屏
  2. {}这两个大括号，作用是上一个下一个视频
  3. 空格键：播放暂停视频
  4. shift+F 全屏 退出屏幕
  5. 上下方向键：调试音量
  6. 左右方向键：快进快退
