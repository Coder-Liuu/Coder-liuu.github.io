解决了Ranger图片不显示的问题

Ranger恢复初始化配置文件的方法,是先删除以前的配置文件，然后进行ranger --copy-config=all 没有使用w3m作为显示图片的工具，而是使用ueberzug,我进行了如下的操作

sudo pacman -S neberzug

接下来修改ranger的配置文件

set preview_images true
set preview_images_method ueberzug
set draw_borders true

linux禁用触摸板

  1. 查看自己所有的输入设备
    xinput list
  2. 找到你触摸板的ID，我这里是13 进行关闭触摸板
    xinput disable 13
  3. 打开触摸板
    xinput enable 13
